# Order Mobile

`ControlerApp/` contains the React Native mobile shell for Order.

`ControlerApp/` 目录是 Order 的 React Native 移动端壳层工程。

## What lives here

- Android and iOS native projects
- React Native runtime and mobile packaging setup
- Synced web assets copied from the root `pages/` directory

## Build entrypoints

- From the repo root, prefer `npm run mobile:install`
- Start Metro with `npm run mobile:start`
- Build Android release APK with `npm run mobile:apk`
- Build iOS release bundle with `npm run mobile:ios`
